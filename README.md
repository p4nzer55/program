
# SaaS Helper - 客服智能问答机器人

基于 saas-security-ops skill 开发的智能客服机器人，提供 SaaS 安全运营相关问题的自动解答。

## 功能特性

- 云监测接入流程咨询
- 玄武盾防护管理问题解答
- 工单处理规范说明
- 常见问题排查（403、422、5XX错误等）
- 平台访问信息查询
- 快捷问题按钮

## 使用方法

### 方式一：直接打开（推荐）

直接用浏览器打开 `chatbot.html` 文件即可使用，无需任何服务器配置。

### 方式二：Node.js 服务器

1. 确保已安装 Node.js
2. 在项目目录运行：
   ```bash
   node index.js
   ```
3. 在浏览器访问：http://localhost:3000

## 文件说明

- `chatbot.html` - 单文件版客服机器人，直接打开即可使用
- `index.html` - 网页版界面（需要配合后端）
- `index.js` - Node.js 后端服务器
- `knowledge-base.js` - 知识库文件
- `package.json` - 项目配置文件
- `.trae/skills/saas-security-ops/SKILL.md` - 原始 skill 文件

## 技术栈

- 纯 HTML/CSS/JavaScript（无需框架）
- 关键词匹配算法
- 响应式设计

## 知识库内容

机器人包含以下类别的问题解答：

1. 问候语
2. 云监测接入
3. 玄武盾防护
4. 防护模式切换
5. 403误拦截处理
6. 422错误排查
7. 5XX错误处理
8. 站点无法访问
9. 工单处理
10. VPN信息
11. 平台访问
12. 永久在线功能
13. 回源/回切
14. 感谢语
15. 告别语

## 扩展开发

如需添加更多问题类型，可以修改 `knowledgeBase` 对象，添加新的关键词和回复内容。
